**Company:** Educational Technology Platform, 2024

**Role:** Full Stack Developer

- Implemented a user-centric dashboard for managing learning paths and certifications
- Built an intuitive interface for displaying user certificates and achievements using Next.js and TypeScript
- Integrated AI-powered recommendation system using OpenAI and Langchain
- Created responsive and accessible UI components using Radix UI and Tailwind CSS
- Optimized performance with TanStack Query for efficient data fetching and caching

**Product**: User Learning Journey Dashboard

Key Features:

- Personal welcome section with user profile
- Certificate showcase with visual representations
- AI-powered instructor recommendations
- Real-time data updates and synchronization
- Secure authentication and authorization
- Clean and modern dark theme interface
- Optimized performance with server-side rendering
- Fully responsive design for mobile and desktop
- Comprehensive instructor profiles and reviews
- Member management dashboard

The dashboard provides learners with a comprehensive view of their educational progress, certificates, and personalized instructor recommendations, enhancing their learning experience through an intuitive and engaging interface powered by modern web technologies and AI.
